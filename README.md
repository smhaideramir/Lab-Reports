# Physics Lab Reports

This repository contains selected physics lab reports from my B.Sc. studies at Leipzig University.

The goal of this repository is to document my experimental work in a clearer and more reproducible way than simply storing final PDF reports. Where available, I include analysis notebooks, generated plots, setup images, and notes about the workflow behind each report.

This repository is still being improved gradually. Some experiments currently include only the final report, while others include additional analysis material.

## Repository structure
