# E1e Plots

This folder contains plots generated during the E1e analysis.

The plots are included separately so that the main results can be viewed quickly without opening the full lab report or notebook.

## Plot list

| File | Description |
|---|---|
| `task1_series_V_vs_R.png` | Voltage versus resistance for the task 1 series circuit analysis |
| `task1_parallel_I_vs_invR.png` | Current versus inverse resistance for the task 1 parallel circuit analysis |
| `task2_UL_vs_RL.png` | Load voltage versus load resistance |
| `task2_UoverUL_vs_invRL.png` | Reciprocal relationship used in task 2 analysis |
| `task3_UB_vs_R4_fit.png` | Task 3 circuit measurement with fitted curve |
| `task4_transfer_HP_measured_vs_theory.png` | Measured and theoretical transfer function for the high-pass filter task |
