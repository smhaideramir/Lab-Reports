# E1e Analysis Notebook

This folder contains the Jupyter notebook used for the E1e lab analysis.

The notebook includes the calculation steps, generated plots, curve fitting, and comparison between measured and theoretical behaviour.

## File

| File | Description |
|---|---|
| `E1e.ipynb` | Main analysis notebook for the E1e experiment |

## How to open

The notebook can be opened with Jupyter Notebook, JupyterLab, or VS Code.

Typical scientific Python packages used for this type of analysis include:

```bash
pip install numpy pandas matplotlib scipy
```

Then open the notebook with:

```bash
jupyter notebook E1e.ipynb
```

## Note

The measurement values are currently included inside the notebook. They have not yet been extracted into separate CSV files.
